# Dayy Reference Palette

## 原始色板

| Token | Value | 用途 |
| --- | --- | --- |
| near-black | `#10131b` | 主文字、页脚大标题、深色首屏 |
| white | `#ffffff` | 主页面背景、深色区域反白文字 |
| soft-gray | `#f3f3f3` | Logo/内容 tile 背景、底部浮动导航 |
| text-subtle | `rgba(16, 19, 27, 0.56)` | 大段说明中的弱化文字 |
| text-muted | `rgba(16, 19, 27, 0.35)` | 次要说明、页脚辅助信息 |
| sunset-accent | `#ff7b2f` | 首屏氛围色，不作为大面积按钮色 |

## 使用规则

- 大面积使用白底和近黑字，靠字号、留白和图片建立品牌感。
- 弱化文字用透明近黑，而不是浅灰色硬编码。
- 圆角主要为 `8px`、`12px` 和 `999px` 三档。
- CTA 和导航不靠高饱和色堆叠，更多依赖黑白对比与悬浮胶囊。
